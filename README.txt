Welcome on my github-page where I post the code that is used in my papers.
Each folder contains the code of one paper:
1) Paper_param_eigv: This code is used in the paper:

Ruymbeek, Koen, Karl Meerbergen, and Wim Michiels. 
"Calculating the minimal/maximal eigenvalue of symmetric parameterized matrices using projection." 
Numerical Linear Algebra with Applications 26.5 (2019): e2263.

2) Paper_Tensor-Krylov:

Ruymbeek, Koen, Karl Meerbergen, and Wim Michiels. 
"Tensor-Krylov method for computing eigenvalues of parameter-dependent matrices." 
arXiv preprint arXiv:2006.07473 (2020).
